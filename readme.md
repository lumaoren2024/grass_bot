开始运行：

1、安装 virtualenv
```bash
pip3 install virtualenv -i https://pypi.tuna.tsinghua.edu.cn/simple/
mkdir myenv
source myenv/bin/activate
```
2、安装组件
```bash
bash pip3.sh
```
3、运行
```bash
python3 grass.py
```
或者
```bash
python3 noproxy_grass.py
```
就是这么简单
